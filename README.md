## Setting Up

### Rails (Backend)

```
cd social_media
```

```
bundle install
```

```
rails db:create
```
```
rails db:migrate
```

```
rails server
```

### Vue (Frontend)

```
cd frontend
```
```
npm install
```

```
npm run serve
```

## Rspec (Testing)

```
cd social_media
```

```
bundle exec rspec
```
